# @openrun/ui

> Licensed under MIT. See [docs/licenses/PLAIN-ENGLISH.md](../../docs/licenses/PLAIN-ENGLISH.md) for what this means for you.

React component library for OpenRun. Provides reusable, accessible UI primitives for displaying coaching data:

- Pace calculators (mile/km conversion, race pace tables)
- Training zone charts (colour-coded effort bands)
- VDOT progress graphs
- Weekly load and acute:chronic workload ratio visualisations
- Race countdown and course profile components

Intentionally licensed under MIT — we want these components to be usable in any running app without license friction.

## Install

```bash
pnpm add @openrun/ui
```

## Development

```bash
pnpm install
pnpm build
pnpm test
```
