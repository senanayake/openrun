# @openrun/mobile

> Licensed under AGPL v3. See [docs/licenses/PLAIN-ENGLISH.md](../../docs/licenses/PLAIN-ENGLISH.md) for what this means for you.

OpenRun mobile app. React Native + Expo, targeting iOS and Android.

- Live workout screen with real-time zone guidance
- Training plan card for the day
- Post-run analysis with effort breakdown
- Wearable sync (Garmin, Apple Watch, WHOOP, Oura)

## Development

```bash
pnpm install
pnpm dev          # opens Expo Go
```

Requires Node 20 and the Expo CLI. For device builds see the Expo documentation.
